# Gulp Basic Workflow

- Basic gulp workflow boilerplate.
- `node >= 20`

## Installation

```
pnpm install
```

## PNPM Scripts

1. Dev task & run dev server + Hot Reload - ```pnpm run start``` ✔️
2. Linting CSS - ```pnpm run lint:css``` ✔️
3. Linting JS - ```pnpm run lint:js``` ✔️
4. Build task - ```pnpm run build``` ✔️
5. Test - ```pnpm test``` ✔️
6. Clean dev folder - ```pnpm run clean:dev``` ✔️
7. Clean build folder - ```pnpm run clean:build``` ✔️
8. Autofix linting issues - ```pnpm run lint:fix``` ️✔️
9. Deploy Project - ```pnpm run deploy``` ❌
